USER GUIDE

A practical workflow guide.

SAT-DISCOVERY helps users organize geospatial resources into a controlled production workflow. It does not replace professional judgement and does not claim to host or own user data.

1

Connect input

Connect user-controlled data or storage. Check file format, CRS, bounds and basic metadata before production.

Open Input Center →
2

Define AOI

Search, draw or import AOI boundaries. Validate geometry before using the AOI in a production job.

Open AOI Tools →
3

Select practical nodes

Choose only production nodes that can be clearly explained and responsibly configured.

Open Production Toolkit →
4

Build job

Combine input, AOI, production nodes, QC requirements and output destination into a job chain.

Open Job Builder →
5

Monitor progress

Track each node, review errors and decide whether to retry, repair inputs or rebuild the job.

Open Progress Monitor →
6

Preview and QC

Use the 2D map workspace to review AOI, output footprint, coverage, offset and QC status.

Open Preview & QC →
7

Prepare delivery

Package outputs, prepare delivery notes, checksums and cleanup confirmation.

Open Output Center →
8

Review manifest

Check the production manifest before sharing final outputs externally.

Open Production Manifest →
Trust-first principle: capabilities shown in SAT-DISCOVERY should be explainable, configurable and responsibly deliverable. Experimental or unverified functions should not be promoted as live production capabilities.

Intelligence may provide options, summaries and reminders, but users and professionals make final decisions. Source notes and disclaimer review should be completed before delivery.
