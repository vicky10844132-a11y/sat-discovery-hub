GEOSPATIAL PRODUCTION CONTROL

Run production without hosting customer data.

SAT-DISCOVERY connects user-controlled data with online geospatial production workflows. It supports AOI definition, processing tool selection, 2D map-based preview and final output delivery.

Core capabilities

Input CenterConnect files and storage.
AOI ToolsSearch, draw, import and validate AOI.
Production ToolkitApply selected processing tools.
Preview & QCCheck coverage, location and offset.
Output CenterPackage, encrypt and deliver outputs.
Orbit PredictionReview pass and coverage position.

Production parameters, task progress and job history are managed inside the production workspace, not on the homepage.
